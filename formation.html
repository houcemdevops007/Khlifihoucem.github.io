Formation intra‑entreprise · 3 jours

Formation DevOps avancé (AWS · Kubernetes · CI/CD · IA)

Un parcours 100% pratique pour amener vos équipes de l’intégration continue aux déploiements automatisés sur Kubernetes, en exploitant GitHub Actions, GitLab CI, Jenkins et l’IA au service du DevOps.

CI/CD multi‑outils Docker & Kubernetes AWS Antigravity / IA Ateliers pratiques

Public & prérequis

Public : développeurs, Ops, DevOps, Tech Leads.

  • Équipes qui ont déjà une CI mais peu de standards.
  • Équipes qui veulent aligner leurs pratiques sur Kubernetes.

Prérequis :

  • Notions de Git et ligne de commande.
  • Connaissances de base sur Docker.

Programme sur 3 jours

Jour 1 – CI/CD multi‑outils

  • Rappels DevOps & CI/CD (objectifs, bonnes pratiques).
  • Concepts clés : GitHub Actions, GitLab CI, Jenkins.
  • Atelier : pipeline de tests sur les 3 outils à partir d’un même projet.

Jour 2 – Docker & Kubernetes

  • Images, registries, tags, bonnes pratiques Docker.
  • Manifests Kubernetes : Deployment, Service, probes, rolling update.
  • Atelier : pipeline complet build → push → déploiement sur Kubernetes.

Jour 3 – IA au service du DevOps

  • Découverte d’Antigravity et cas d’usage DevOps.
  • Atelier : génération / amélioration de pipelines avec l’IA.
  • Focus sécurité : scans, secrets, monitoring & pistes d’industrialisation.

Objectifs pédagogiques

  • Savoir concevoir une chaîne CI/CD robuste et lisible.
  • Savoir déployer une application conteneurisée sur Kubernetes en sécurité.
  • Comprendre comment l’IA peut accélérer les tâches DevOps sans perdre le contrôle.
